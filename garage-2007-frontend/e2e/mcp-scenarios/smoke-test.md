# Smoke Test — Garage 2007

Дымовой тест: базовая проверка работоспособности приложения.

## Подготовка
Убедитесь что dev-сервер запущен: `npm run dev` в `garage-2007-frontend/`

## Шаги

1. **Открыть приложение**
   - `browser_navigate("http://localhost:5173")`

2. **Дождаться загрузки**
   - `browser_wait_for("text=DEV")` (DEV overlay появляется после загрузки)
   - Если появился экран "Загрузка..." — подождать его исчезновения

3. **Сделать скриншот начального состояния**
   - `browser_take_screenshot()`
   - Ожидаемо: заголовок с балансом, 4 таба, canvas с гаражом

4. **Проверить структуру DOM**
   - `browser_snapshot()`
   - Ожидаемо: присутствует `canvas`, элементы навигации, DEV overlay с "B: 0"

5. **Проверить наличие табов**
   - Убедиться что видны тексты: "Игра", "Улучшения", "Ачивки", "Статистика"

6. **Проверить DEV overlay**
   - Должен показывать: "DEV", "B: 0", "L: 1"

## Критерии прохождения
- [ ] Приложение открывается без ошибок
- [ ] Загрузка завершается (нет "Загрузка...")
- [ ] Canvas отрисован
- [ ] Все 4 таба видимы
- [ ] DEV overlay показывает корректные значения
