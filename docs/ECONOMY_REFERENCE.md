## Формулы

### Стоимость апгрейда
```
Cost(n) = floor(BaseCost × 1.15^n)
```

### Доход за клик
```
ClickIncome = (clickPowerLevel + 1) × CritMultiplier × BoostMultiplier
CritMultiplier = 2.0 (шанс 5%) или 1.0
```

### Пассивный доход
```
BasePassive = Σ(worker.count × WORKER_INCOME[type])
PassivePerSec = BasePassive × (1.0 + workSpeedLevel × 0.1) × BoostMultiplier
```

### Оффлайн-доход
```
OfflineIncome = PassivePerSec × EffectiveTime
EffectiveTime:
  0-8ч  → 100% (28800 сек)
  8-24ч → 50%
  >24ч  → cap
Минимум: 60 сек отсутствия
```
